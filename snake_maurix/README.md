## 🐍 Snake Android (HTML5)

> Juego clásico de Snake optimizado para dispositivos móviles Android.

**Archivo:** `snake_android.html`

### Características
- 📱 Controles táctiles con D-Pad en pantalla
- 👆 Gestos de deslizamiento (swipe) para cambiar dirección
- ⌨️ Compatible con teclado (flechas + Espacio para pausar)
- 🏆 Sistema de puntuación con mejor puntaje guardado localmente
- 📈 Niveles progresivos — la serpiente acelera con cada nivel
- ⏸️ Botón de pausa
- 🎨 Diseño retro con efecto neón

### Cómo jugar
1. Descarga o clona este repositorio
2. Abre `snake_android.html` en el navegador de tu Android (Chrome recomendado)
3. *(Opcional)* En Chrome → **Añadir a pantalla de inicio** para usarlo como app

### Controles
| Acción | Método |
|--------|--------|
| Mover | D-Pad táctil o deslizar el dedo |
| Pausar | Botón ⏸ o tecla `Espacio` |
| Reanudar | Botón ▶ o tecla `Espacio` |

## 🐍 Snake Python

> Juego clásico de Snake para escritorio con Python 3.

**Archivo:** `snake.py`

### Requisitos
```bash
pip install pygame
```

### Cómo ejecutar
```bash
python snake.py
```

### Controles
| Acción | Tecla |
|--------|-------|
| Mover | Flechas del teclado |
| Pausar | `P` o `Espacio` |
| Salir | `ESC` o cerrar ventana |

## 🚀 Instalación rápida

```bash
# Clonar el repositorio
git clone https://github.com/maurixmystic/mis-juegos.git
cd mis-juegos

# Para el Snake Python, instalar dependencia
pip install pygame

# Ejecutar Snake Python
python snake.py

# Para Snake Android, simplemente abre snake_android.html en el navegador
```
