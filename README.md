# 🎮 mis-juegos

> Colección de juegos clásicos y retro programados por **[@maurixmystic](https://github.com/maurixmystic)** — con algo de magia de la IA de por medio.

---

## 🕹️ ¿De qué trata este repositorio?

Aquí iré subiendo juegos que programo por mi cuenta y también con ayuda de Inteligencia Artificial. El foco estará en los clásicos que marcaron generaciones: simples, adictivos y con mucho carácter retro.

## 🛠️ Tecnologías

Los juegos pueden estar hechos con distintas tecnologías según el proyecto:

- **HTML + CSS + JavaScript** (canvas, vanilla JS)
- **Python** (pygame u otras librerías)
- Otras herramientas según el juego

---

## 🤖 Rol de la IA

Algunos juegos son desarrollados con ayuda de herramientas de Inteligencia Artificial como asistente de programación. La IA colabora en la lógica, el código y la resolución de problemas — pero la dirección, las ideas y las decisiones son mías.

---

## 🚀 Cómo ejecutar los juegos

Cada juego tendrá su propia carpeta con un `README` interno que explica cómo correrlo. En general:

```bash
# Clonar el repositorio
git clone https://github.com/maurixmystic/mis-juegos.git

# Entrar a la carpeta del juego
cd mis-juegos/nombre-del-juego

# Seguir las instrucciones del README de cada juego
```
